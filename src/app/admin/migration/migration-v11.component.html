v11 升级指南

本文档仅作为 ng-antd-admin 升级 angular 11 示例。

开始之前

1. 首先确保你 Node.js >= 10.13

2. 创建新的分支，或者使用其他方式备份当前项目

3. 删除项目下 package-lock.json 文件

升级步骤

1. 升级相关依赖

  • 前往 https://update.angular.io/ 将项目升级到 Angular 11
  • 如果你有单独使用 @angular/cdk 请执行 ng update @angular/cdk

2. 升级 NG-ZORRO

  • 运行 ng update ng-zorro-antd
  • 如果控制台出现警告消息请按提示修改对应代码

本项目升级步骤

1. 将项目升级到 Angular 11

  • 运行 ng update @angular/cli@11 @angular/core@11

2. 将 @angular/cdk 升级到 11

  • 运行 npm i --save @angular/cdk@11.x

3. 将 ng-zorro-antd 升级到 11

  • 运行 npm i --save ng-zorro-antd@11.x
